## Requirements / Dependencies
* Arch
`sudo pacman -S nodejs npm luarocks`
* Debian
`sudo apt install nodejs npm luarocks`
* Gentoo
`sudo emerge -va nodejs npm yarn luarocks`

## Installation
`git clone https://github.com/zakky20/nvim.git` & `sudo mv nvim ~/.config`
